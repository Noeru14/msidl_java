# ORGANISATION DU DOSSIER
- img : dossier stockant les images utilisées dans le programme ;
- save : dossier stockant les évolutions sauvegardées, il permet également de charger ces - mêmes évolutions ;
- conf : dossier stockant les configurations initiales.

# FONCTIONNALITES PRINCIPALES
- définir une population initiale manuellement ;
- définir une population initiale aléatoire ; 
- charger une configuration initiale ; 

# FONCTIONNALITES COMPLEMENTAIRES (selon cahier des charges)
- sauvegarder et charger une évolution en cours ; 
- coloration des cellules.
